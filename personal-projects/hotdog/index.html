Hotdog Regression

2024-09-06 ~ 2024-09-07
September 6, 2024

SUMMARY#

A simple hotdog classification web application inspired by the hotdog/not-hotdog idea from HBO’s Silicon Valley.

GOAL#

Understand how FastAPI builds API servers and interacts with ML model.

PARTICIPANT#

Alone

ROLE#

Data engineering, Frontend, Backend 

TOOLS#

  • Language: Python
  • Forntend: Streamlit
  • Backent: FastAPI
  • ML: Hotdog model by HuggingFace

ARCHITECTURE#

Project architecture

POINT#

  • Generating hotdog discrimination results. FastAPI - main.py
    • Load ‘Julien-c/hotdog-not-hotdog’, a pre-trained model of the Hugging Face.
    • ‘Image.open (io.BytesIO(img)’ converts images received in bytes into PIL image objects.
    • ‘p[0][‘score’]’ takes the reliability score predicted by the model and returns the result to the client in JSON format.

OUTPUT#

if it is a hotdog 

if it is not a hotdog
