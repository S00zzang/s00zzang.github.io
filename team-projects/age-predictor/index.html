    Age Classfier

    2024-10-08 ~ 2024-10-10
    October 8, 2024

    SUMMARY#

    A program that predicts a person’s age using an Age Classifier model when a face photo is uploaded.

    GOAL#

    Create an internal management program that manages services for ML and DL models, aggregates data, and analyzes it.

    PARTICIPANT#

    5 (AC)

    ROLE#

    Backend(Image preprocessing), Data Analysis

    TOOLS#

    • Frontend: Streamlit
    • Backend: FastAPI
    • Workflow Management: Apache Airflow
    • Data Analysis and Visualization: Apache Spark
    • DB: Maria DB      
    • ML: age-classifier model by HuggingFace

    ARCHITECTURE#

    Project architecture

    POINT#

    • Key Flows for File Upload and DB Interworking FastAPI - main.py
      • sA structure that takes this file path from the Airflow DAG, executes the prediction, and stores the prediction results back in the DB.
      • This logic is used to process file uploads, and then FastAPI and Airflow are linked to work sequentially.

    OUTPUT#

    age prediction result
