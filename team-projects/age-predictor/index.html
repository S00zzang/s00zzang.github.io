SUMMARY#

A program that predicts a person’s age using an Age Classifier model when a face photo is uploaded.

GOAL#

Create an internal management program that manages services for ML and DL models, aggregates data, and analyzes it.

PARTICIPANT#

5 (AC)

ROLE#

Backend(Image preprocessing), Data Analysis

TOOLS#

  • Backend: FastAPI
  • Workflow Management: Apache Airflow
  • Data Analysis and Visualization: Apache Spark
  • DB: Maria DB      
  • ML: age-classifier model by HuggingFace
